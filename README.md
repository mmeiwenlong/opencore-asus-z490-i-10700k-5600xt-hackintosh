## Hardware

* Motherboard: Asus ROG STRIX Z490-I
* CPU: Intel® Core™ i7-10700K
* GPU: Sapphire Pulse Radeon RX 5600 XT 6GB DDR6
* RAM: CORSAIR VENGEANCE LPX DDR4 2666 16GB(8x2)

* Bootloader: OpenCore 0.5.9
* OS: macOS Catalina 10.15.6
